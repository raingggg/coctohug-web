ติดตั้ง

  • เปิดเทอร์มินัลและเปลี่ยนไดเร็กทอรีการทำงานเป็นโฟลเดอร์ที่มี ccm.sh
  • (Linux / Mac) เริ่ม ccm โดยสร้างลิงค์: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

ช่วย

  • คู่มือคำสั่ง: ccm help

 

คำสั่งตัวอย่างสำหรับส้อม blockchain ฟลอร่า

  • ติดตั้งส้อมบล็อคเชนหนึ่งอัน เช่น ฟลอรา: ccm install flora
  • เริ่มต้นหนึ่ง blockchain fork เช่น ฟลอรา: ccm start flora
  • หยุดหนึ่งส้อมบล็อคเชนเช่นฟลอรา: ccm stop flora
  • รีสตาร์ทหนึ่งส้อมบล็อคเชนเช่นฟลอรา: ccm restart flora
  • อัปเกรดส้อมบล็อคเชนหนึ่งอัน เช่น ฟลอรา: ccm upgrade flora
  • ถอนการติดตั้ง blockchain fork หนึ่งอัน เช่น ฟลอรา: ccm uninstall flora

 

คำสั่งด่วนสำหรับส้อมบล็อคเชนทั้งหมด

  • ติดตั้งส้อม blockchain ทั้งหมด: ccm install all
  • เริ่มส้อม blockchain ทั้งหมด: ccm start all
  • หยุดส้อมบล็อคเชนทั้งหมด: ccm stop all
  • รีสตาร์ทส้อมบล็อคเชนทั้งหมด: ccm restart all
  • อัปเกรดส้อม blockchain ทั้งหมด: ccm upgrade all
  • ถอนการติดตั้ง blockchain fork ทั้งหมด: ccm uninstall all

 

ดูสถานะ fork blockchain โดยละเอียด

  • ดูข้อมูลการเชื่อมต่อเช่น ethgreen: ccm vconnection ethgreen
  • ดูข้อมูลสรุปฟาร์มเช่น ethgreen: ccm vsummary ethgreen
  • ดูข้อมูลกระเป๋าเงินเช่น ethgreen: ccm vwallet ethgreen
  • ดูข้อมูลคีย์เช่น ethgreen: ccm vkey ethgreen
  • ดูข้อมูลบันทึกเช่น ethgreen: ccm vlog ethgreen

 

ไปที่คอนเทนเนอร์นักเทียบท่า

  • เข้าไปข้างในนักเทียบท่าเช่น tranzact: ccm docker tranzact

 

การจัดการการปฏิบัติงานของนักเทียบท่า

  • ล้างภาพนักเทียบท่าที่ไม่ได้ใช้ทั้งหมด: ccm clean
  • แสดงคอนเทนเนอร์นักเทียบท่าทั้งหมด: ccm container
  • แสดงภาพนักเทียบท่าทั้งหมด: ccm image
