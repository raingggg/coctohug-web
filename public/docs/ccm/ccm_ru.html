Настраивать

  • Откройте терминал и измените рабочий каталог на папку, содержащую ccm.sh
  • (Linux / Mac) Запустите ccm, создав ссылку: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Помощь

  • Руководство по командам: ccm help

 

Примеры команд для форка блокчейна флора

  • Установите одну вилку блокчейна, такую ​​как флора: ccm install flora
  • Запустите один форк блокчейна, такой как флора: ccm start flora
  • Остановите одну вилку блокчейна, такую ​​как флора: ccm stop flora
  • Перезапустите одну вилку блокчейна, такую ​​как флора: ccm restart flora
  • Обновите одну вилку блокчейна, такую ​​как флора: ccm upgrade flora
  • Удалите одну вилку блокчейна, такую ​​как флора: ccm uninstall flora

 

Быстрые команды для всех форков блокчейна

  • Установите все форки блокчейна: ccm install all
  • Запустите все форки блокчейна: ccm start all
  • Остановите все форки блокчейна: ccm stop all
  • Перезапустите все форки блокчейна: ccm restart all
  • Обновите все форки блокчейна: ccm upgrade all
  • Удалите все форки блокчейна: ccm uninstall all

 

Просмотр подробного статуса форка блокчейна

  • Просмотрите информацию о подключении, например, ethgreen: ccm vconnection ethgreen
  • Просмотр сводной информации о ферме, например, ethgreen: ccm vsummary ethgreen
  • Просмотрите информацию о кошельке, такую ​​как ethgreen: ccm vwallet ethgreen
  • Просмотрите информацию о ключах, такую ​​как ethgreen: ccm vkey ethgreen
  • Просмотрите информацию журнала, такую ​​как ethgreen: ccm vlog ethgreen

 

Перейти в контейнер докеров

  • Зайдите внутрь докера, такого как tranzact: ccm docker tranzact

 

Оперативное управление Docker

  • Очистите все неиспользуемые образы докеров: ccm clean
  • Показать все докер-контейнеры: ccm container
  • Показать все образы Docker: ccm image
