Настройвам

  • Отворете терминала и променете работната директория в папката, която съдържа ccm.sh
  • (Linux / Mac) Инициирайте ccm чрез създаване на връзка: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Помогне

  • Ръководство за команди: ccm help

 

Примерни команди за флора блокчейн вилица

  • Инсталирайте една блокчейн вилка, като например флора: ccm install flora
  • Стартирайте една блокчейн вилка, като флора: ccm start flora
  • Спрете една блокчейн вилка, като флора: ccm stop flora
  • Рестартирайте една блокчейн вилка, като например флора: ccm restart flora
  • Надстройте една блокчейн вилка, като например флора: ccm upgrade flora
  • Деинсталирайте една блокчейн вилка, като flora: ccm uninstall flora

 

Бързи команди за всички блокчейн разклонения

  • Инсталирайте всички блокчейн разклонения: ccm install all
  • Стартирайте всички блокчейн разклонения: ccm start all
  • Спрете всички блокчейн разклонения: ccm stop all
  • Рестартирайте всички блокчейн разклонения: ccm restart all
  • Надстройте всички блокчейн разклонения: ccm upgrade all
  • Деинсталирайте всички блокчейн разклонения: ccm uninstall all

 

Вижте подробно състояние на блокчейн форк

  • Вижте информация за връзката като ethgreen: ccm vconnection ethgreen
  • Вижте обобщена информация за фермата, като ethgreen: ccm vsummary ethgreen
  • Вижте информация за портфейла като ethgreen: ccm vwallet ethgreen
  • Вижте информация за ключовете като ethgreen: ccm vkey ethgreen
  • Вижте информация за регистрационните файлове, като ethgreen: ccm vlog ethgreen

 

Отидете до контейнера за докер

  • Влезте в докер като tranzact: ccm docker tranzact

 

Оперативно управление на Docker

  • Почистете всички неизползвани докер изображения: ccm clean
  • Показване на всички докер контейнери: ccm container
  • Показване на всички изображения на Docker: ccm image
