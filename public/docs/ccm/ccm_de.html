Konfiguration

  • Öffnen Sie das Terminal und ändern Sie das Arbeitsverzeichnis in den Ordner, der ccm.sh enthält
  • (Linux / Mac)Init ccm durch Erstellen eines Links: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Hilfe

  • Befehlsanleitung: ccm help

 

Beispielbefehle für die Flora-Blockchain-Fork

  • Installieren Sie eine Blockchain-Fork wie Flora: ccm install flora
  • Starten Sie eine Blockchain-Fork wie Flora: ccm start flora
  • Stoppen Sie eine Blockchain-Gabel wie Flora: ccm stop flora
  • Starten Sie eine Blockchain-Fork wie Flora neu: ccm restart flora
  • Aktualisieren Sie eine Blockchain-Gabel wie Flora: ccm upgrade flora
  • Deinstallieren Sie eine Blockchain-Fork wie Flora: ccm uninstall flora

 

Schnellbefehle für alle Blockchain-Gabeln

  • Installieren Sie alle Blockchain-Forks: ccm install all
  • Starten Sie alle Blockchain-Forks: ccm start all
  • Stoppen Sie alle Blockchain-Forks: ccm stop all
  • Starten Sie alle Blockchain-Forks neu: ccm restart all
  • Aktualisieren Sie alle Blockchain-Forks: ccm upgrade all
  • Deinstallieren Sie alle Blockchain-Forks: ccm uninstall all

 

Detaillierten Blockchain-Fork-Status anzeigen

  • Verbindungsinformationen wie Ethgreen anzeigen: ccm vconnection ethgreen
  • Anzeigen der Farmzusammenfassungsinformationen wie Ethgreen: ccm vsummary ethgreen
  • Anzeigen von Wallet-Informationen wie Ethgreen: ccm vwallet ethgreen
  • Anzeigen von Schlüsselinformationen wie Ethgreen: ccm vkey ethgreen
  • Log-Informationen wie Ethgreen anzeigen: ccm vlog ethgreen

 

Gehe zum Docker-Container

  • Gehen Sie in Docker wie tranzact: ccm docker tranzact

 

Docker-Betriebsmanagement

  • Bereinigen Sie alle nicht verwendeten Docker-Images: ccm clean
  • Alle Docker-Container anzeigen: ccm container
  • Alle Docker-Images anzeigen: ccm image
