設定

  • ターミナルを開き、作業ディレクトリをccm.shを含むフォルダに変更します
  • （Linux / Mac）リンクを作成してccmを初期化します： sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

ヘルプ

  • コマンドガイド： ccm help

 

フローラブロックチェーンフォークのサンプルコマンド

  • 植物相などの1つのブロックチェーンフォークをインストールします。 ccm install flora
  • 植物相などの1つのブロックチェーンフォークを開始します。 ccm start flora
  • 植物相などの1つのブロックチェーンフォークを停止します。 ccm stop flora
  • 植物相などの1つのブロックチェーンフォークを再起動します。 ccm restart flora
  • 植物相などの1つのブロックチェーンフォークをアップグレードします。 ccm upgrade flora
  • アップグレードしてから、フローラなどの1つのブロックチェーンフォークを開始します。 ccm upup flora
  • 植物相などの1つのブロックチェーンフォークをアンインストールします。 ccm uninstall flora

 

すべてのブロックチェーンフォークのクイックコマンド

  • すべてのブロックチェーンフォークをインストールします。 ccm install all
  • すべてのブロックチェーンフォークを開始します。 ccm start all
  • すべてのブロックチェーンフォークを停止します。 ccm stop all
  • すべてのブロックチェーンフォークを再起動します。 ccm restart all
  • すべてのブロックチェーンフォークをアップグレードします。 ccm upgrade all
  • アップグレードしてから、すべてのブロックチェーンフォークを開始します。 ccm upup all
  • すべてのブロックチェーンフォークをアンインストールします。 ccm uninstall all

 

詳細なブロックチェーンフォークのステータスを表示する

  • ethgreenなどの接続情報を表示します。 ccm vconnection ethgreen
  • ethgreenなどのファームの概要情報を表示します。 ccm vsummary ethgreen
  • ethgreenなどのウォレット情報を表示します。 ccm vwallet ethgreen
  • ethgreenなどのキー情報を表示します。 ccm vkey ethgreen
  • ethgreenなどのログ情報を表示します。 ccm vlog ethgreen

 

Dockerコンテナに移動します

  • tranzactなどのDocker内に移動します。 ccm docker tranzact

 

Dockerの運用管理

  • 未使用のDockerイメージをすべてクリーンアップします。 ccm clean
  • すべてのDockerコンテナを表示します。 ccm container
  • すべてのDockerイメージを表示します。 ccm image
