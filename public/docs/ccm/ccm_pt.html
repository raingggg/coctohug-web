Configurar

  • Abra o terminal e altere o diretório de trabalho para a pasta que contém ccm.sh
  • (Linux / Mac) Inicie o ccm criando o link: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Ajuda

  • Guia de comando: ccm help

 

Comandos de amostra para o fork do blockchain flora

  • Instale um fork do blockchain, como flora: ccm install flora
  • Comece um fork do blockchain, como flora: ccm start flora
  • Pare um fork do blockchain, como flora: ccm stop flora
  • Reinicie um fork do blockchain, como flora: ccm restart flora
  • Atualize um fork do blockchain, como flora: ccm upgrade flora
  • Desinstale um fork do blockchain, como flora: ccm uninstall flora

 

Comandos rápidos para todos os garfos blockchain

  • Instale todos os garfos blockchain: ccm install all
  • Inicie todos os garfos do blockchain: ccm start all
  • Pare todos os garfos do blockchain: ccm stop all
  • Reinicie todos os garfos do blockchain: ccm restart all
  • Atualize todos os garfos blockchain: ccm upgrade all
  • Desinstale todos os garfos do blockchain: ccm uninstall all

 

Ver o status detalhado do fork do blockchain

  • Visualize informações de conexão, como ethgreen: ccm vconnection ethgreen
  • Ver informações de resumo da fazenda, como ethgreen: ccm vsummary ethgreen
  • Veja as informações da carteira, como a ethgreen: ccm vwallet ethgreen
  • Ver informações de chaves, como ethgreen: ccm vkey ethgreen
  • Ver informações de registro, como ethgreen: ccm vlog ethgreen

 

Vá para o contêiner do docker

  • Vá para dentro do docker, como tranzact: ccm docker tranzact

 

Gestão operacional Docker

  • Limpe todas as imagens do docker não utilizadas: ccm clean
  • Mostrar todos os contêineres do docker: ccm container
  • Mostrar todas as imagens do Docker: ccm image
