Montar

  • Abre o terminal e cambia o directorio de traballo ao cartafol que contén ccm.sh
  • (Linux/Mac)Iniciar ccm creando unha ligazón: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Axuda

  • Guía de comandos: ccm help

 

Comandos de exemplo para o fork da cadea de bloques flora

  • Instala un fork de cadea de bloques como flora: ccm install flora
  • Inicia unha bifurcación de cadea de bloques como flora: ccm start flora
  • Detén un garfo de cadea de bloques como flora: ccm stop flora
  • Reinicie unha bifurcación de cadea de bloques como flora: ccm restart flora
  • Actualiza un fork de cadea de bloques como flora: ccm upgrade flora
  • Desinstale un fork de cadea de bloques como flora: ccm uninstall flora

 

Comandos rápidos para todos os forks de blockchain

  • Instala todos os forks blockchain: ccm install all
  • Inicia todos os forks de blockchain: ccm start all
  • Detén todas as bifurcacións de blockchain: ccm stop all
  • Reinicia todos os forks de blockchain: ccm restart all
  • Actualiza todos os forks de blockchain: ccm upgrade all
  • Desinstale todos os forks de blockchain: ccm uninstall all

 

Consulta o estado detallado do fork blockchain

  • Ver información de conexión como ethgreen: ccm vconnection ethgreen
  • Ver información de resumo da granxa como ethgreen: ccm vsummary ethgreen
  • Ver información da carteira como ethgreen: ccm vwallet ethgreen
  • Ver información de claves como ethgreen: ccm vkey ethgreen
  • Ver información de rexistro como ethgreen: ccm vlog ethgreen

 

Vaia ao contedor docker

  • Vaia dentro do docker como transacción: ccm docker tranzact

 

Xestión operativa de Docker

  • Limpar todas as imaxes docker non utilizadas: ccm clean
  • Mostrar todos os contedores docker: ccm container
  • Mostra todas as imaxes de Docker: ccm image
