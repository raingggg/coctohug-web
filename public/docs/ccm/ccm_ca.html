Configuració

  • Obriu el terminal i canvieu el directori de treball a la carpeta que conté ccm.sh
  • (Linux / Mac)Inicieu ccm creant un enllaç: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Ajuda

  • Guia de comandaments: ccm help

 

Comandes d'exemple per a la bifurcació de cadena de blocs de flora

  • Instal·leu una bifurcació de cadena de blocs, com ara flora: ccm install flora
  • Inicieu una bifurcació de cadena de blocs, com ara flora: ccm start flora
  • Atureu una bifurcació de cadena de blocs com la flora: ccm stop flora
  • Reinicieu una bifurcació de cadena de blocs, com ara flora: ccm restart flora
  • Actualitzeu una bifurcació de cadena de blocs, com ara flora: ccm upgrade flora
  • Actualitzeu i inicieu una bifurcació de cadena de blocs, com ara flora: ccm upup flora
  • Desinstal·leu una bifurcació de cadena de blocs, com ara flora: ccm uninstall flora

 

Ordres ràpides per a totes les bifurcacions de blockchain

  • Instal·leu totes les bifurcacions blockchain: ccm install all
  • Inicieu totes les bifurcacions de blockchain: ccm start all
  • Atura totes les bifurcacions de blockchain: ccm stop all
  • Reinicieu totes les bifurcacions de blockchain: ccm restart all
  • Actualitzeu totes les bifurcacions de blockchain: ccm upgrade all
  • Actualitzeu i inicieu totes les bifurcacions de blockchain: ccm upup all
  • Desinstal·leu totes les bifurcacions blockchain: ccm uninstall all

 

Mostra l'estat detallat de la forquilla de blockchain

  • Veure informació de connexió com ara ethgreen: ccm vconnection ethgreen
  • Veure informació de resum de la granja, com ara ethgreen: ccm vsummary ethgreen
  • Veure informació de la cartera com ara ethgreen: ccm vwallet ethgreen
  • Veure la informació de les claus com ara ethgreen: ccm vkey ethgreen
  • Veure informació de registre com ara ethgreen: ccm vlog ethgreen

 

Vés al contenidor Docker

  • Aneu dins del docker com ara transaccions: ccm docker tranzact

 

Gestió operativa de Docker

  • Netegeu totes les imatges de Docker no utilitzades: ccm clean
  • Mostra tots els contenidors Docker: ccm container
  • Mostra totes les imatges de Docker: ccm image
