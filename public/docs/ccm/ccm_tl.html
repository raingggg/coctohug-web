Setup

  • Buksan ang terminal at baguhin ang gumaganang direktoryo sa folder na naglalaman ng ccm.sh
  • (Linux / Mac)Init ccm sa pamamagitan ng paggawa ng link: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Tulong

  • Gabay sa command: ccm help

 

Mga halimbawang utos para sa flora blockchain fork

  • Mag-install ng isang blockchain fork gaya ng flora: ccm install flora
  • Magsimula ng isang blockchain fork tulad ng flora: ccm start flora
  • Itigil ang isang blockchain fork gaya ng flora: ccm stop flora
  • I-restart ang isang blockchain fork gaya ng flora: ccm restart flora
  • Mag-upgrade ng isang blockchain fork gaya ng flora: ccm upgrade flora
  • Mag-upgrade pagkatapos magsimula ng isang blockchain fork gaya ng flora: ccm upup flora
  • I-uninstall ang isang blockchain fork gaya ng flora: ccm uninstall flora

 

Mabilis na utos para sa lahat ng blockchain forks

  • I-install ang lahat ng blockchain forks: ccm install all
  • Simulan ang lahat ng blockchain forks: ccm start all
  • Itigil ang lahat ng blockchain forks: ccm stop all
  • I-restart ang lahat ng blockchain forks: ccm restart all
  • I-upgrade ang lahat ng blockchain forks: ccm upgrade all
  • Mag-upgrade pagkatapos simulan ang lahat ng blockchain forks: ccm upup all
  • I-uninstall ang lahat ng blockchain forks: ccm uninstall all

 

Tingnan ang detalyadong blockchain fork status

  • Tingnan ang impormasyon ng koneksyon gaya ng ethgreen: ccm vconnection ethgreen
  • Tingnan ang buod ng impormasyon sa bukid gaya ng ethgreen: ccm vsummary ethgreen
  • Tingnan ang impormasyon ng wallet gaya ng ethgreen: ccm vwallet ethgreen
  • Tingnan ang impormasyon ng mga key gaya ng ethgreen: ccm vkey ethgreen
  • Tingnan ang impormasyon ng log gaya ng ethgreen: ccm vlog ethgreen

 

Pumunta sa docker container

  • Pumunta sa loob ng docker tulad ng tranzact: ccm docker tranzact

 

Pamamahala ng pagpapatakbo ng Docker

  • Linisin ang lahat ng hindi nagamit na larawan ng docker: ccm clean
  • Ipakita ang lahat ng docker container: ccm container
  • Ipakita ang lahat ng larawan ng Docker: ccm image
