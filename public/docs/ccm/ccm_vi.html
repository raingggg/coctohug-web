Thành lập

  • Mở terminal và thay đổi thư mục làm việc thành thư mục chứa ccm.sh
  • (Linux / Mac) Init ccm bằng cách tạo liên kết: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Cứu giúp

  • Hướng dẫn lệnh: ccm help

 

Các lệnh mẫu cho fork blockchain thực vật

  • Cài đặt một nhánh blockchain như hệ thực vật: ccm install flora
  • Bắt đầu một nhánh blockchain chẳng hạn như hệ thực vật: ccm start flora
  • Dừng một nhánh blockchain chẳng hạn như hệ thực vật: ccm stop flora
  • Khởi động lại một nhánh blockchain chẳng hạn như hệ thực vật: ccm restart flora
  • Nâng cấp một nhánh blockchain như hệ thực vật: ccm upgrade flora
  • Gỡ cài đặt một nhánh blockchain như hệ thực vật: ccm uninstall flora

 

Các lệnh nhanh cho tất cả các nhánh blockchain

  • Cài đặt tất cả các nhánh blockchain: ccm install all
  • Bắt đầu tất cả các nhánh blockchain: ccm start all
  • Dừng tất cả các nhánh blockchain: ccm stop all
  • Khởi động lại tất cả các nhánh blockchain: ccm restart all
  • Nâng cấp tất cả các nhánh blockchain: ccm upgrade all
  • Gỡ cài đặt tất cả các nhánh blockchain: ccm uninstall all

 

Xem trạng thái fork blockchain chi tiết

  • Xem thông tin kết nối chẳng hạn như ethgreen: ccm vconnection ethgreen
  • Xem thông tin tóm tắt về trang trại chẳng hạn như ethgreen: ccm vsummary ethgreen
  • Xem thông tin ví như ethgreen: ccm vwallet ethgreen
  • Xem thông tin khóa chẳng hạn như ethgreen: ccm vkey ethgreen
  • Xem thông tin nhật ký chẳng hạn như ethgreen: ccm vlog ethgreen

 

Đi tới vùng chứa docker

  • Đi vào bên trong docker chẳng hạn như tranzact: ccm docker tranzact

 

Quản lý hoạt động Docker

  • Làm sạch tất cả các hình ảnh docker không sử dụng: ccm clean
  • Hiển thị tất cả các vùng chứa docker: ccm container
  • Hiển thị tất cả hình ảnh Docker: ccm image
