يثبت

  • افتح Terminal وقم بتغيير دليل العمل إلى المجلد الذي يحتوي على ccm.sh
  • (Linux / Mac) ابدأ ccm بإنشاء رابط: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

يساعد

  • دليل القيادة: ccm help

 

أوامر عينة لشوكة بلوكشين فلورا

  • قم بتثبيت مفترق blockchain واحد مثل flora: ccm install flora
  • ابدأ شوكة blockchain واحدة مثل flora: ccm start flora
  • أوقف شوكة واحدة من سلسلة الكتل مثل فلورا: ccm stop flora
  • أعد تشغيل شوكة blockchain واحدة مثل flora: ccm restart flora
  • قم بترقية مفترق blockchain واحد مثل flora: ccm upgrade flora
  • قم بإلغاء تثبيت مفترق blockchain واحد مثل flora: ccm uninstall flora

 

أوامر سريعة لجميع مفترقات blockchain

  • قم بتثبيت جميع مفترقات blockchain: ccm install all
  • ابدأ كل شوكات blockchain: ccm start all
  • أوقف جميع شوكات blockchain: ccm stop all
  • أعد تشغيل جميع مفترقات blockchain: ccm restart all
  • قم بترقية جميع شوكات blockchain: ccm upgrade all
  • قم بإلغاء تثبيت جميع مفترقات blockchain: ccm uninstall all

 

عرض مفصل لحالة شوكة blockchain

  • عرض معلومات الاتصال مثل ethgreen: ccm vconnection ethgreen
  • عرض معلومات ملخص المزرعة مثل ethgreen: ccm vsummary ethgreen
  • عرض معلومات المحفظة مثل ethgreen: ccm vwallet ethgreen
  • عرض معلومات المفاتيح مثل ethgreen: ccm vkey ethgreen
  • عرض معلومات السجل مثل ethgreen: ccm vlog ethgreen

 

اذهب إلى حاوية الرصيف

  • اذهب داخل عامل بناء مثل ترانزاكت: ccm docker tranzact

 

إدارة تشغيل عامل ميناء

  • تنظيف كافة صور عامل الإرساء غير المستخدمة: ccm clean
  • عرض كل حاويات الرصيف: ccm container
  • عرض كل صور Docker: ccm image
