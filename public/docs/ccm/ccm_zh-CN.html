设置

  • 打开终端并将工作目录更改为包含 ccm.sh 的文件夹
  • (Linux / Mac) 通过创建链接初始化 ccm： sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

帮助

  • 命令指南： ccm help

 

Flora 区块链分叉的示例命令

  • 安装一个区块链分叉，例如 flora： ccm install flora
  • 启动一个区块链分叉，例如 flora： ccm start flora
  • 停止一个区块链分叉，例如 flora： ccm stop flora
  • 重启一个区块链分叉，例如 flora： ccm restart flora
  • 升级一个区块链分叉，例如 flora： ccm upgrade flora
  • 卸载一个区块链分叉，例如 flora： ccm uninstall flora

 

所有区块链分叉的快速命令

  • 安装所有区块链分叉： ccm install all
  • 启动所有区块链分叉： ccm start all
  • 停止所有区块链分叉： ccm stop all
  • 重启所有区块链分叉： ccm restart all
  • 升级所有区块链分叉： ccm upgrade all
  • 卸载所有区块链分叉： ccm uninstall all

 

查看详细的区块链分叉状态

  • 查看 ethgreen 等连接信息： ccm vconnection ethgreen
  • 查看农场摘要信息，例如 ethgreen： ccm vsummary ethgreen
  • 查看ethgreen等钱包信息： ccm vwallet ethgreen
  • 查看 ethgreen 等密钥信息： ccm vkey ethgreen
  • 查看 ethgreen 等日志信息： ccm vlog ethgreen

 

转到 docker 容器

  • 进入 docker，例如 tranzact： ccm docker tranzact

 

Docker 运营管理

  • 清理所有未使用的 docker 镜像： ccm clean
  • 显示所有 docker 容器： ccm container
  • 显示所有 Docker 镜像： ccm image
