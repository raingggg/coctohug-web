설정

  • 터미널을 열고 작업 디렉토리를 ccm.sh가 포함된 폴더로 변경합니다.
  • (Linux / Mac)링크를 생성하여 ccm 초기화: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

돕다

  • 명령 가이드: ccm help

 

식물상 블록체인 포크에 대한 샘플 명령

  • 식물상과 같은 하나의 블록체인 포크를 설치합니다. ccm install flora
  • 식물상과 같은 하나의 블록체인 포크 시작: ccm start flora
  • 식물상과 같은 하나의 블록체인 포크를 중지합니다. ccm stop flora
  • 식물상과 같은 하나의 블록체인 포크를 다시 시작합니다. ccm restart flora
  • 식물상과 같은 하나의 블록체인 포크 업그레이드: ccm upgrade flora
  • 식물상과 같은 하나의 블록체인 포크를 제거합니다. ccm uninstall flora

 

모든 블록체인 포크에 대한 빠른 명령

  • 모든 블록체인 포크 설치: ccm install all
  • 모든 블록체인 포크 시작: ccm start all
  • 모든 블록체인 포크 중지: ccm stop all
  • 모든 블록체인 포크를 다시 시작합니다. ccm restart all
  • 모든 블록체인 포크 업그레이드: ccm upgrade all
  • 모든 블록체인 포크 제거: ccm uninstall all

 

자세한 블록체인 포크 상태 보기

  • ethgreen과 같은 연결 정보 보기: ccm vconnection ethgreen
  • ethgreen과 같은 팜 요약 정보 보기: ccm vsummary ethgreen
  • ethgreen과 같은 지갑 정보 보기: ccm vwallet ethgreen
  • ethgreen과 같은 키 정보 보기: ccm vkey ethgreen
  • ethgreen과 같은 로그 정보 보기: ccm vlog ethgreen

 

도커 컨테이너로 이동

  • tranzact와 같은 도커 내부로 이동합니다. ccm docker tranzact

 

도커 운영 관리

  • 사용하지 않는 모든 도커 이미지를 정리합니다. ccm clean
  • 모든 도커 컨테이너 표시: ccm container
  • 모든 Docker 이미지 표시: ccm image
