Подесити

  • Отворите терминал и промените радни директоријум у фасциклу која садржи ццм.сх
  • (Линук / Мац) Покрените ццм креирањем везе: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Помоћ

  • Водич за команде: ccm help

 

Пример команди за флора блоцкцхаин виљушку

  • Инсталирајте једну блоцкцхаин виљушку као што је флора: ccm install flora
  • Покрените једну блоцкцхаин виљушку као што је флора: ccm start flora
  • Зауставите једну блокчејн виљушку као што је флора: ccm stop flora
  • Поново покрените једну блоцкцхаин виљушку као што је флора: ccm restart flora
  • Надоградите једну блоцкцхаин виљушку као што је флора: ccm upgrade flora
  • Надоградите, а затим покрените једну блокчејн виљушку као што је флора: ccm upup flora
  • Деинсталирајте једну блоцкцхаин виљушку као што је флора: ccm uninstall flora

 

Брзе команде за све блокчејн виљушке

  • Инсталирајте све блоцкцхаин виљушке: ccm install all
  • Покрените све блокчејн виљушке: ccm start all
  • Зауставите све блокчејн виљушке: ccm stop all
  • Поново покрените све блокчејн виљушке: ccm restart all
  • Надоградите све блоцкцхаин виљушке: ccm upgrade all
  • Надоградите, а затим покрените све блокчејн виљушке: ccm upup all
  • Деинсталирајте све блоцкцхаин виљушке: ccm uninstall all

 

Погледајте детаљан статус блоцкцхаин форк

  • Погледајте информације о вези као што је етхгреен: ccm vconnection ethgreen
  • Погледајте сажетак информација о фарми као што је етхгреен: ccm vsummary ethgreen
  • Погледајте информације о новчанику као што је етхгреен: ccm vwallet ethgreen
  • Погледајте информације о кључевима као што је етхгреен: ccm vkey ethgreen
  • Погледајте информације из дневника као што је етхгреен: ccm vlog ethgreen

 

Идите на Доцкер контејнер

  • Идите у доцкер као што је транзацт: ccm docker tranzact

 

Доцкер оперативни менаџмент

  • Очистите све некоришћене доцкер слике: ccm clean
  • Прикажи све доцкер контејнере: ccm container
  • Прикажи све Доцкер слике: ccm image
