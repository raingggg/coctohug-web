Kurmak

  • Terminali açın ve çalışma dizinini ccm.sh içeren klasöre değiştirin
  • (Linux / Mac) Bağlantı oluşturarak ccm'yi başlatın: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Yardım

  • Komut kılavuzu: ccm help

 

Flora blok zinciri çatalı için örnek komutlar

  • Flora gibi bir blockchain çatalı kurun: ccm install flora
  • Flora gibi bir blockchain çatalı başlatın: ccm start flora
  • Flora gibi bir blockchain çatalını durdurun: ccm stop flora
  • Flora gibi bir blockchain çatalını yeniden başlatın: ccm restart flora
  • Flora gibi bir blockchain çatalını yükseltin: ccm upgrade flora
  • Yükseltme ve ardından flora gibi bir blok zinciri çatalı başlatın: ccm upup flora
  • Flora gibi bir blockchain çatalını kaldırın: ccm uninstall flora

 

Tüm blockchain çatalları için hızlı komutlar

  • Tüm blockchain çatallarını kurun: ccm install all
  • Tüm blockchain çatallarını başlatın: ccm start all
  • Tüm blockchain çatallarını durdurun: ccm stop all
  • Tüm blockchain çatallarını yeniden başlatın: ccm restart all
  • Tüm blockchain çatallarını yükseltin: ccm upgrade all
  • Yükselt ve ardından tüm blockchain çatallarını başlat: ccm upup all
  • Tüm blockchain çatallarını kaldırın: ccm uninstall all

 

Ayrıntılı blockchain çatal durumunu görüntüleyin

  • ethgreen gibi bağlantı bilgilerini görüntüleyin: ccm vconnection ethgreen
  • ethgreen gibi çiftlik özet bilgilerini görüntüleyin: ccm vsummary ethgreen
  • ethgreen gibi cüzdan bilgilerini görüntüleyin: ccm vwallet ethgreen
  • ethgreen gibi anahtar bilgilerini görüntüleyin: ccm vkey ethgreen
  • ethgreen gibi günlük bilgilerini görüntüleyin: ccm vlog ethgreen

 

liman işçisi konteynerine git

  • Transzact gibi docker'ın içine girin: ccm docker tranzact

 

Docker operasyon yönetimi

  • Kullanılmayan tüm liman işçisi görüntülerini temizleyin: ccm clean
  • Tüm liman işçisi kapsayıcılarını göster: ccm container
  • Tüm Docker resimlerini göster: ccm image
