Installer

  • Ouvrez le terminal et remplacez le répertoire de travail par le dossier contenant ccm.sh
  • (Linux / Mac)Initez ccm en créant un lien : sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Aider

  • Guide de commande : ccm help

 

Exemples de commandes pour le fork de la blockchain flora

  • Installez un fork blockchain tel que flora : ccm install flora
  • Démarrez un fork blockchain tel que flora : ccm start flora
  • Arrêtez une fourchette de blockchain telle que flora : ccm stop flora
  • Redémarrez un fork blockchain tel que flora : ccm restart flora
  • Mettez à niveau une fourchette de blockchain telle que flora : ccm upgrade flora
  • Désinstallez un fork blockchain tel que flora : ccm uninstall flora

 

Commandes rapides pour tous les forks blockchain

  • Installez toutes les fourches blockchain : ccm install all
  • Démarrez tous les forks blockchain : ccm start all
  • Arrêtez toutes les fourches blockchain : ccm stop all
  • Redémarrez tous les forks blockchain : ccm restart all
  • Mettez à niveau toutes les fourches blockchain : ccm upgrade all
  • Désinstallez toutes les fourches blockchain : ccm uninstall all

 

Afficher le statut détaillé de la fourche blockchain

  • Afficher les informations de connexion telles que ethgreen : ccm vconnection ethgreen
  • Affichez les informations de synthèse de la ferme telles que ethgreen : ccm vsummary ethgreen
  • Afficher les informations du portefeuille telles que ethgreen : ccm vwallet ethgreen
  • Afficher les informations sur les clés telles que ethgreen : ccm vkey ethgreen
  • Afficher les informations de journal telles que ethgreen : ccm vlog ethgreen

 

Aller au conteneur docker

  • Allez à l'intérieur de docker tel que transzact : ccm docker tranzact

 

Gestion opérationnelle de Docker

  • Nettoyez toutes les images Docker inutilisées : ccm clean
  • Afficher tous les conteneurs Docker : ccm container
  • Afficher toutes les images Docker : ccm image
