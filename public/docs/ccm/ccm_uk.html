Налаштування

  • Відкрийте термінал і змініть робочий каталог на папку, що містить ccm.sh
  • (Linux / Mac) Ініціювати ccm шляхом створення посилання: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Допоможіть

  • Посібник з командами: ccm help

 

Приклади команд для блокчейн-вилки flora

  • Встановіть один блокчейн-форк, наприклад flora: ccm install flora
  • Запустіть один блокчейн-форк, наприклад flora: ccm start flora
  • Зупиніть один блокчейн-форк, такий як флора: ccm stop flora
  • Перезапустіть один блокчейн-форк, наприклад flora: ccm restart flora
  • Оновіть один блокчейн-форк, наприклад flora: ccm upgrade flora
  • Оновіть, а потім запустіть один блокчейн-форк, наприклад flora: ccm upup flora
  • Видаліть один блокчейн-форк, наприклад flora: ccm uninstall flora

 

Швидкі команди для всіх форків блокчейну

  • Встановіть усі блокчейн-форки: ccm install all
  • Запустіть усі форки блокчейну: ccm start all
  • Зупинити всі форки блокчейну: ccm stop all
  • Перезапустіть усі блокчейн-форки: ccm restart all
  • Оновіть усі блокчейн-форки: ccm upgrade all
  • Оновіть, а потім запустіть усі блокчейн-форки: ccm upup all
  • Видаліть усі блокчейн-форки: ccm uninstall all

 

Перегляньте детальний статус форка блокчейну

  • Перегляд інформації про підключення, наприклад ethgreen: ccm vconnection ethgreen
  • Перегляд підсумкової інформації про ферму, наприклад ethgreen: ccm vsummary ethgreen
  • Перегляд інформації про гаманець, наприклад ethgreen: ccm vwallet ethgreen
  • Перегляд інформації про ключі, наприклад ethgreen: ccm vkey ethgreen
  • Перегляд інформації журналу, наприклад ethgreen: ccm vlog ethgreen

 

Перейдіть до контейнера докерів

  • Зайдіть всередину докера, наприклад tranzact: ccm docker tranzact

 

Операційне управління Docker

  • Очистіть усі невикористані образи Docker: ccm clean
  • Показати всі контейнери Docker: ccm container
  • Показати всі зображення Docker: ccm image
