Impostare

  • Apri il terminale e cambia la directory di lavoro nella cartella contiene ccm.sh
  • (Linux/Mac)Inizia il ccm creando il collegamento: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Aiuto

  • Guida ai comandi: ccm help

 

Comandi di esempio per il fork della blockchain flora

  • Installa un fork di blockchain come flora: ccm install flora
  • Avvia un fork di blockchain come flora: ccm start flora
  • Ferma un fork di blockchain come flora: ccm stop flora
  • Riavvia un fork di blockchain come flora: ccm restart flora
  • Aggiorna un fork di blockchain come flora: ccm upgrade flora
  • Disinstalla un fork di blockchain come flora: ccm uninstall flora

 

Comandi rapidi per tutti i fork blockchain

  • Installa tutti i fork blockchain: ccm install all
  • Avvia tutti i fork blockchain: ccm start all
  • Interrompi tutti i fork della blockchain: ccm stop all
  • Riavvia tutti i fork della blockchain: ccm restart all
  • Aggiorna tutti i fork blockchain: ccm upgrade all
  • Disinstalla tutti i fork blockchain: ccm uninstall all

 

Visualizza lo stato dettagliato del fork blockchain

  • Visualizza le informazioni di connessione come ethgreen: ccm vconnection ethgreen
  • Visualizza le informazioni di riepilogo della fattoria come ethgreen: ccm vsummary ethgreen
  • Visualizza le informazioni sul portafoglio come ethgreen: ccm vwallet ethgreen
  • Visualizza le informazioni sulle chiavi come ethgreen: ccm vkey ethgreen
  • Visualizza le informazioni di registro come ethgreen: ccm vlog ethgreen

 

Vai al container docker

  • Vai all'interno della finestra mobile come tranzact: ccm docker tranzact

 

Gestione operativa Docker

  • Pulisci tutte le immagini docker inutilizzate: ccm clean
  • Mostra tutti i contenitori docker: ccm container
  • Mostra tutte le immagini Docker: ccm image
