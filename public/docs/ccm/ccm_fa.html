برپایی

  • ترمینال را باز کنید و دایرکتوری کاری را به پوشه حاوی ccm.sh تغییر دهید
  • (Linux / Mac) با ایجاد پیوند، ccm را راه اندازی کنید: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

کمک

  • راهنمای فرمان: ccm help

 

نمونه دستورات برای فورک بلاک چین فلورا

  • یک فورک بلاک چین مانند فلورا نصب کنید: ccm install flora
  • یک فورک بلاک چین مانند فلورا را راه اندازی کنید: ccm start flora
  • یک فورک بلاک چین مانند فلور را متوقف کنید: ccm stop flora
  • یک فورک بلاک چین مانند فلورا را مجددا راه اندازی کنید: ccm restart flora
  • یک فورک بلاک چین مانند فلورا را ارتقا دهید: ccm upgrade flora
  • ارتقا دهید و سپس یک فورک بلاک چین مانند فلورا را شروع کنید: ccm upup flora
  • یک فورک بلاک چین مانند فلورا را حذف نصب کنید: ccm uninstall flora

 

دستورات سریع برای همه فورک های بلاک چین

  • همه فورک های بلاک چین را نصب کنید: ccm install all
  • همه فورک های بلاک چین را راه اندازی کنید: ccm start all
  • تمام فورک های بلاک چین را متوقف کنید: ccm stop all
  • همه فورک های بلاک چین را مجددا راه اندازی کنید: ccm restart all
  • تمام فورک های بلاک چین را ارتقا دهید: ccm upgrade all
  • ارتقا دهید و سپس تمام فورک های بلاک چین را راه اندازی کنید: ccm upup all
  • تمام فورک های بلاک چین را حذف کنید: ccm uninstall all

 

وضعیت جزئیات فورک بلاک چین را مشاهده کنید

  • مشاهده اطلاعات اتصال مانند ethgreen: ccm vconnection ethgreen
  • مشاهده اطلاعات خلاصه مزرعه مانند ethgreen: ccm vsummary ethgreen
  • مشاهده اطلاعات کیف پول مانند ethgreen: ccm vwallet ethgreen
  • مشاهده اطلاعات کلیدها مانند ethgreen: ccm vkey ethgreen
  • مشاهده اطلاعات گزارش مانند ethgreen: ccm vlog ethgreen

 

به کانتینر داکر بروید

  • به داخل داکر مانند tranzact بروید: ccm docker tranzact

 

مدیریت عملیاتی داکر

  • تمام تصاویر داکر استفاده نشده را پاک کنید: ccm clean
  • نمایش همه کانتینرهای docker: ccm container
  • نمایش تمام تصاویر داکر: ccm image
