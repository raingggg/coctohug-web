設置

  • 打開終端並將工作目錄更改為包含 ccm.sh 的文件夾
  • (Linux / Mac) 通過創建鏈接初始化 ccm： sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

幫助

  • 命令指南： ccm help

 

Flora 區塊鏈分叉的示例命令

  • 安裝一個區塊鏈分叉，例如 flora： ccm install flora
  • 啟動一個區塊鏈分叉，例如 flora： ccm start flora
  • 停止一個區塊鏈分叉，例如 flora： ccm stop flora
  • 重啟一個區塊鏈分叉，例如 flora： ccm restart flora
  • 升級一個區塊鏈分叉，例如 flora： ccm upgrade flora
  • 升級然後啟動一個區塊鏈分叉，例如flora： ccm upup flora
  • 卸載一個區塊鏈分叉，例如 flora： ccm uninstall flora

 

所有區塊鏈分叉的快速命令

  • 安裝所有區塊鏈分叉： ccm install all
  • 啟動所有區塊鏈分叉： ccm start all
  • 停止所有區塊鏈分叉： ccm stop all
  • 重啟所有區塊鏈分叉： ccm restart all
  • 升級所有區塊鏈分叉： ccm upgrade all
  • 升級然後啟動所有區塊鏈分叉： ccm upup all
  • 卸載所有區塊鏈分叉： ccm uninstall all

 

查看詳細的區塊鏈分叉狀態

  • 查看 ethgreen 等連接信息： ccm vconnection ethgreen
  • 查看農場摘要信息，例如 ethgreen： ccm vsummary ethgreen
  • 查看ethgreen等錢包信息： ccm vwallet ethgreen
  • 查看 ethgreen 等密鑰信息： ccm vkey ethgreen
  • 查看 ethgreen 等日誌信息： ccm vlog ethgreen

 

轉到 docker 容器

  • 進入 docker，例如 tranzact： ccm docker tranzact

 

Docker 運營管理

  • 清理所有未使用的 docker 鏡像： ccm clean
  • 顯示所有 docker 容器： ccm container
  • 顯示所有 Docker 鏡像： ccm image
