Înființat

  • Deschideți terminalul și schimbați directorul de lucru în folderul care conține ccm.sh
  • (Linux / Mac) Inițiază ccm prin crearea linkului: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Ajutor

  • Ghid de comandă: ccm help

 

Exemple de comenzi pentru furculița blockchain flora

  • Instalați o furcă blockchain, cum ar fi flora: ccm install flora
  • Porniți o furcă blockchain, cum ar fi flora: ccm start flora
  • Opriți o furcă blockchain, cum ar fi flora: ccm stop flora
  • Reporniți o furcă blockchain, cum ar fi flora: ccm restart flora
  • Actualizați o furcă blockchain, cum ar fi flora: ccm upgrade flora
  • Faceți upgrade, apoi porniți un bloc blockchain, cum ar fi flora: ccm upup flora
  • Dezinstalați o furcă blockchain, cum ar fi flora: ccm uninstall flora

 

Comenzi rapide pentru toate furcile blockchain

  • Instalați toate furcile blockchain: ccm install all
  • Porniți toate furcile blockchain: ccm start all
  • Opriți toate furcile blockchain: ccm stop all
  • Reporniți toate furcile blockchain: ccm restart all
  • Actualizați toate furcile blockchain: ccm upgrade all
  • Faceți upgrade, apoi porniți toate forkurile blockchain: ccm upup all
  • Dezinstalați toate furcile blockchain: ccm uninstall all

 

Vezi starea detaliată a blockchain fork

  • Vedeți informații despre conexiune, cum ar fi ethgreen: ccm vconnection ethgreen
  • Vedeți informațiile rezumate ale fermei, cum ar fi ethgreen: ccm vsummary ethgreen
  • Vedeți informații despre portofel, cum ar fi ethgreen: ccm vwallet ethgreen
  • Vizualizați informații despre chei, cum ar fi ethgreen: ccm vkey ethgreen
  • Vedeți informațiile din jurnal, cum ar fi ethgreen: ccm vlog ethgreen

 

Accesați containerul docker

  • Intrați în docker, cum ar fi tranzact: ccm docker tranzact

 

Management operațional Docker

  • Curățați toate imaginile docker neutilizate: ccm clean
  • Afișați toate containerele docker: ccm container
  • Afișați toate imaginile Docker: ccm image
