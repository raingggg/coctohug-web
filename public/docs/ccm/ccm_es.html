Configuración

  • Abra la terminal y cambie el directorio de trabajo a la carpeta que contiene ccm.sh
  • (Linux / Mac) Inicie ccm creando un enlace: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Ayudar

  • Guía de comando: ccm help

 

Comandos de muestra para la bifurcación de la cadena de bloques de flora

  • Instale una bifurcación de blockchain como flora: ccm install flora
  • Inicie una bifurcación de blockchain como flora: ccm start flora
  • Detén una bifurcación de blockchain como flora: ccm stop flora
  • Reinicie una bifurcación de blockchain como flora: ccm restart flora
  • Actualice una bifurcación de blockchain como flora: ccm upgrade flora
  • Desinstale una bifurcación de blockchain como flora: ccm uninstall flora

 

Comandos rápidos para todas las bifurcaciones de blockchain

  • Instale todas las bifurcaciones de blockchain: ccm install all
  • Inicie todas las bifurcaciones de blockchain: ccm start all
  • Detenga todas las bifurcaciones de blockchain: ccm stop all
  • Reinicie todas las bifurcaciones de blockchain: ccm restart all
  • Actualice todas las bifurcaciones de blockchain: ccm upgrade all
  • Desinstale todas las bifurcaciones de blockchain: ccm uninstall all

 

Ver el estado detallado de la bifurcación de blockchain

  • Ver información de conexión como ethgreen: ccm vconnection ethgreen
  • Ver información de resumen de la granja, como ethgreen: ccm vsummary ethgreen
  • Ver información de billetera como ethgreen: ccm vwallet ethgreen
  • Ver información de claves como ethgreen: ccm vkey ethgreen
  • Ver información de registro como ethgreen: ccm vlog ethgreen

 

Ir al contenedor de la ventana acoplable

  • Vaya dentro de la ventana acoplable como tranzact: ccm docker tranzact

 

Gestión operativa de Docker

  • Limpiar todas las imágenes de la ventana acoplable no utilizadas: ccm clean
  • Mostrar todos los contenedores Docker: ccm container
  • Mostrar todas las imágenes de Docker: ccm image
