Mulai cepat

  • Mempersiapkan
  • Membantu

  • Perintah cepat untuk semua garpu blockchain

Mempersiapkan

  • Buka terminal dan ubah direktori kerja ke folder berisi ccm.sh
  • (Linux / Mac)Init ccm dengan membuat tautan: sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

Membantu

  • Panduan perintah: ccm help

 

Contoh perintah untuk garpu blockchain flora

  • Instal satu garpu blockchain seperti flora: ccm install flora
  • Mulai satu garpu blockchain seperti flora: ccm start flora
  • Hentikan satu garpu blockchain seperti flora: ccm stop flora
  • Mulai ulang satu garpu blockchain seperti flora: ccm restart flora
  • Tingkatkan satu garpu blockchain seperti flora: ccm upgrade flora
  • Tingkatkan lalu mulai satu garpu blockchain seperti flora: ccm upup flora
  • Copot satu garpu blockchain seperti flora: ccm uninstall flora

 

Perintah cepat untuk semua garpu blockchain

  • Instal semua garpu blockchain: ccm install all
  • Mulai semua garpu blockchain: ccm start all
  • Hentikan semua garpu blockchain: ccm stop all
  • Mulai ulang semua garpu blockchain: ccm restart all
  • Tingkatkan semua garpu blockchain: ccm upgrade all
  • Tingkatkan lalu mulai semua garpu blockchain: ccm upup all
  • Copot semua garpu blockchain: ccm uninstall all

 

Lihat status garpu blockchain yang terperinci

  • Lihat info koneksi seperti ethgreen: ccm vconnection ethgreen
  • Lihat info ringkasan pertanian seperti ethgreen: ccm vsummary ethgreen
  • Lihat info dompet seperti ethgreen: ccm vwallet ethgreen
  • Lihat info kunci seperti ethgreen: ccm vkey ethgreen
  • Lihat info log seperti ethgreen: ccm vlog ethgreen

 

Pergi ke wadah buruh pelabuhan

  • Masuk ke dalam buruh pelabuhan seperti tranzact: ccm docker tranzact

 

Manajemen operasional buruh pelabuhan

  • Bersihkan semua gambar buruh pelabuhan yang tidak digunakan: ccm clean
  • Tampilkan semua wadah buruh pelabuhan: ccm container
  • Tampilkan semua gambar Docker: ccm image
