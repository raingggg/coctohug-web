Quick Start

 

-

  • sudo ln -s /full/path/to/your/file/ccm.sh /usr/local/bin/ccm

 

  • ccm help

 

  • ccm install flora
  • ccm start flora
  • ccm stop flora
  • ccm restart flora
  • ccm upgrade flora
  • ccm uninstall flora

 

  • ccm install all
  • ccm start all
  • ccm stop all
  • ccm restart all
  • ccm upgrade all
  • ccm uninstall all

 

  • ccm vconnection ethgreen
  • ccm vsummary ethgreen
  • ccm vwallet ethgreen
  • ccm vkey ethgreen
  • ccm vlog ethgreen

 

  • ccm docker tranzact

 

  • ccm clean
  • ccm container
  • ccm image
